# JSON Explorer Documentation

JSON Explorer is an interactive web tool that analyzes and visualizes the structure of JSON data, whether uploaded as a file or pasted directly. It generates a comprehensive overview including a collapsible tree view of nested paths, a sortable and filterable table of all paths with type and occurrence information, statistical summaries for numeric and string fields, and an exportable markdown report. The tool handles large files gracefully with capped analysis at 200,000 nodes and includes features like tree search, value distribution visualization in a bottom drawer, URL-based state persistence for table filters and sorts, and one-click copying of JSON paths for reference.
