# JSON Explorer

The JSON Explorer is an interactive tool for analyzing and exploring the structure of JSON files and data. Users can upload JSON files or paste JSON text directly to generate comprehensive reports including an overview of data statistics, an expandable tree visualization of the document structure, a complete list of all data paths with type information and occurrence counts, and a markdown-formatted report suitable for documentation or sharing. The tool supports JSONL format, provides inline statistics such as value distributions and numeric summaries, and allows navigation through deeply nested structures with customizable expansion controls for efficient exploration of complex datasets.
